Notification Block Structure Example

Correct Structure for Notification Item

Each notification item should have these fields in order:

1. Message (richtext)
Don't miss out! Get up to 8.55%*p.a. and grow your savings faster
2. Call to Action Text (text)
Invest Now
3. Call to Action Link (aem-content)
<a href="https://www.shriramfinance.in/fixed-deposit">Invest Now</a>
4. Timestamp (text)
2 hours ago

Expected HTML Output

After processing, the notification item should look like:

<li class="notification-item">
  <div class="notification-message">
    <p>Don't miss out! Get up to 8.55%*p.a. and grow your savings faster</p>
  </div>
  <div class="notification-call-to-action">
    <a href="https://www.shriramfinance.in/fixed-deposit">Invest Now</a>
  </div>
  <div class="notification-timestamp">
    <p>2 hours ago</p>
  </div>
</li>

Content Author Instructions

When creating notification items in AEM:

  1. Message: Add the main notification text (richtext field)
  2. Call to Action Text: Add the button/link text (e.g., "Invest Now")
  3. Call to Action Link: Add the actual link/button (aem-content field)
  4. Timestamp: Add when the notification was created (e.g., "2 hours ago")
